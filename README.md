
# 🌐 Digital E Gram Panchayat

A comprehensive web-based platform designed to digitize rural governance, enabling **citizens**, **staff**, and **admin** to manage scheme applications and approvals efficiently.

*Live Demo : spectacular-cat-7d0ee4.netlify.app
*Unique Id Admin :JH2025ADMIN
*Unique Id Staff : JH2025STAFF

## 🚀 Features

### ✅ **User Module**

* **Profile Management:** Update and maintain personal information and upload profile picture.
* **Available Schemes:** View and apply for government schemes.
* **Application Tracking:** Monitor real-time application status and history.

### ✅ **Staff Module**

* **Scheme Management:** Delete, and manage schemes.
* **Application Processing:** Review, approve, or reject user applications.
* **Forwarding:** Forward approved applications to admin for final review.

### ✅ **Admin Module**

* **Final Approval:** Approve or reject applications after staff review.
* **Scheme Oversight:** Manage all schemes and monitor system-wide activity.
* **Role-based Control:** Secure access based on user role (User, Staff, Admin).

---

## 💻 Technologies Used

* **Frontend:** React.js (functional components, hooks)
* **UI Framework:** React-Bootstrap / Material UI
* **Authentication:** Firebase Authentication
* **Database:** Firebase Firestore
* **File Storage:** Cloudinary (for image uploads)
* **Real-time Updates:** Firestore `onSnapshot` for live data display



## ⚡️ Workflows

* **User applies** for a scheme.
* **Staff reviews** and forwards to **Admin**.
* **Admin approves/rejects** application.
* Real-time updates for all stakeholders.
